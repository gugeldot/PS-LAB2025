# Number tycoon
This project was developed as part of an academic assignment for the Software patterns subject.

The project focuses on applying object-oriented design principles and multiple software design patterns in a real, functional application.

--- 

## Game Description
Number Tycoon is a real-time automation and strategy game where the player builds and manages a production network based on numerical flows.

The player must extract, transform, and consume numbers using mines, conveyors, mathematical modules, and wells in order to generate points and expand their infrastructure.
